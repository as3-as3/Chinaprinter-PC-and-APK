# Downloads

The ready‑to‑use binaries are **not** stored in the git repository (to keep it lean).

➡️ Get the latest **APK** (Android) and **portable Windows build** from the
**[Releases page](../../releases/latest)**:

- `MXW01-Drucker.apk` — Android app (Android 8+)
- `MXW01-Drucker-Portable.zip` — standalone Windows program (no Python needed)

To build them yourself, see the main [README](../README.md) → *Build from source*.
