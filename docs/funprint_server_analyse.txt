FAZIT
------------------------------------------------------------
Eine simple Etiketten-Drucker-App brachte gleich VIER
Werbe-/Tracking-Netzwerke mit (ByteDance/Pangle, TikTok,
Google Ads, Facebook), dazu Cloud-OCR und mehrere
Analytics-Dienste. Beim bloßen Drucken eines Etiketts
konnten potenziell Nutzungsdaten an all diese Server fließen.

Die selbst gebaute App (PC + Handy):
  - kontaktiert NULL Server
  - kein Tracking, keine Werbung, keine Cloud
  - nur Bluetooth direkt zum Drucker
  - 5,86 MB (Handy) statt 185 MB

wäre per Netzwerk-Mitschnitt (Proxy/tcpdump) möglich.
